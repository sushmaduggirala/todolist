# todolist
A simple web-based TodoList application to manage your tasks.

## Features

- Add new tasks
- Mark tasks as completed
- View the list of tasks

## Technologies Used

- HTML
- CSS
- JavaScript

## Getting Started

### Prerequisites

Make sure you have a modern web browser installed.
